Useful Tips When Writing Codes on Vscode

Author
 2023-01-03  59 words   One minute 
Contents
  1. Zip a folder: zip -r destination_folder_name.zip source_folder_name
  2. Unzip a folder: unzip source_folder_name.zip -d ./destination_folder_name/. Specifically, ./ means current directory, ../ means parent directory, ~ means home directory, / means root directory. Do not use / directory !!!
  3. In linux, terminal, etc., create a new file, touch XXX.cpp
  4. In terminal, command + T will help you open a new window
