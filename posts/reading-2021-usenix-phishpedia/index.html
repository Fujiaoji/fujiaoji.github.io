Research Question

To address two prominent technical challenges in phishing identification, i.e., (i) accurate recognition of identity logos on webpage screenshots, and (ii) matching logo variants of the same brands.

How

Dividing the phishing identification problem into 1) identify logo recognition [use the object detection] and 2) brand recognition problem [use Siamese model].

  • Object detection: generates a set of bounding boxes to annotate the position and size of the objects on the image. Using the Fast RCNN model. Input the screenshot, output the (i) predict the object class (i.e., logo or input box) and (ii) refine shape and size of each object.
  • Brand recognition problem: target brand list, maintain a list of logo variants and a list of legitimate domains. For logo comparison, they use Siamese neural network model (for the Siamese model, input conventional training procedure for Siamese model selects three images (I1, I2, I3) as a sample to calculate Triplet losswhere I1 and I2 belong to same class, and I3 belongs to another class. The goal of training is to make sure the similarity of images in the same class (sim(I1, I2)) should be larger than that in different classes (sim(I2, I3 ))., there are some related papers in this area !!!).
