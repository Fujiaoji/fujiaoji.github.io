Reading: Learning Icons Appearance Similarity

Author
 2023-02-24  302 words   2 minutes 
Contents
    • Reference

What:

  1. main goal is to obtain a similarity metric between a pair of icons
  2. Present an icon dataset labeled by designers where each collection shares a coherent style and visual identity.
  3. We learn icons’ appearance similarity using a Siamese Neural Network with a triplet loss function and adaptive sampling trained from our weakly-labeled dataset and evaluated with human ratings.
  4. We propose several applications including search by similarity and a method to create icon sets optimized for style and visual identity in order to help users on user-interface design tasks.
  5. We collect annotated ratings on the perception of appearance similarity for iconography
  6. Sum: Presented a model for measuring the properties of style and visual identity in iconography. Specifically, trained a Siamese Neural Network with an on-line dataset of icons organized in visually coherent collections that are used to adaptively sample training data and optimize the training process. As the dataset contains noise, we further collect human-rated information on the perception of icon’s similarity which will be used for evaluating and testing the proposed model
  7. Our learned model maps each icon into a 256-dimensional feature space which allows direct comparisons by computing Euclidean distances How
  8. Our main goal is to obtain a similarity metric between a pair of icons
  9. To train similarity metric, they used a weakly labeled dataset. Part of the dataset is used to launch crowde-sourcing surveys and gathering human-rating to test and compare the model. The other part is used to train the Siamese Neural Network to work as the similarity metric. Other Information
